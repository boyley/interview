1. **打基础**：先过 `01-cheatsheet` 的 01→04（Java/集合/并发/JVM），这是 Java 后端必问核心。
2. **补存储**：05 MySQL、06 Redis —— 几乎必问。
3. **框架**：07 Spring。
4. **手撕**：并行刷 `02-coding`，面试现场高频。
5. **进阶**：08~10（网络/OS/分布式）+ `03-system-design`，冲中高级/大厂。
6. **收尾**：`04-project-hr` 打磨项目话术，面前一天过。
